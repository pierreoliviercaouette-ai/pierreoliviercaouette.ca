**Hors portée de ce guide :** RBC Direct Investing (courtage autonome).

---

## Document clé

Avec le relevé annuel de fonds, RBC fournit notamment :

- **Annual Performance Report** — rendement des fonds du compte après frais  
- **Annual Charges and Compensation Report** — coûts (RFG, etc.)

---

## A) Voir les placements (Online Banking)

1. Se connecter à **RBC Online Banking**  
2. Page **Accounts Summary** → cliquer le compte d’investissement / fonds  
3. Consulter soldes, historique, performance affichée  

App mobile : soldes de placements visibles ; pour les PDF, souvent **More → Statements**.

---

## B) Télécharger les documents (texte officiel RBC)

1. Online Banking → **Statements/Documents** (haut de l’écran)  
2. Onglet eDocuments → choisir le **compte** dans le menu  
3. **Go**  
4. Type de document → relevé / **Annual Performance Report** / documents de placements  

App : **More** → **Statements** → choisir le compte.

---

## Quoi envoyer

- PDF **Annual Performance Report** (préférable)  
- Nom du fonds RBC / PH&N
