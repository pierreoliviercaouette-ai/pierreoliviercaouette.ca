# Guide officiel — Relevés et rendements (fonds en banque)

Orienté **placements gérés / fonds communs en succursale ou banque en ligne**  
(AccèsD, EasyWeb, Scotia OnLine, RBC Online Banking, etc.).

**Pas** le courtage autonome (iTRADE, Investor’s Edge, NégociTitres, BMO Ligne d’action, BNCD).

## Limite

Les écrans **après connexion** ne sont pas capturables sans compte.  
Quand la banque publie un **exemple de relevé** ou un **guide PDF** sur ses fonds, on l’utilise.
