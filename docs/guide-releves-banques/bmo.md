# BMO — Fonds communs achetés en succursale / banque en ligne

Sources :
- Page BMO Mutual Funds (achat en succursale ou Online Banking)  
- Documents / relevés dans **BMO Online Banking**

**Hors portée de ce guide :** BMO Ligne d’action (InvestorLine), adviceDirect, app BMO Invest (courtage / conseils numériques autonomes).

---

## Contexte

Si vos fonds BMO ont été achetés **en succursale** ou via **BMO Online Banking**, les relevés et rapports de rendement se consultent dans la banque en ligne (pas dans l’app de courtage).

BMO précise (rapports de fonds) :  
*If you purchased BMO Mutual Funds through a BMO Bank of Montreal branch or BMO Online Banking, please call 1-800-665-7700.*

---

## Parcours pratique

1. Se connecter à **BMO Online Banking**  
2. Ouvrir le compte de **fonds / placements**  
3. Section **Documents / Statements / eStatements**  
4. Télécharger :
   - le relevé de placements  
   - le **rapport annuel de rendement** / Investment Report (souvent avec le relevé de décembre)  
   - le rapport des frais (CRM2), s’il est joint  

---

## Quoi envoyer

- Nom du fonds BMO  
- PDF du relevé + section **taux de rendement** (money-weighted / personnel)
