# Quoi noter sur le relevé

Avant d’envoyer une capture ou un PDF, cochez :

## Informations essentielles

- [ ] Nom exact du **fonds ou portefeuille**
- [ ] Type de compte (REER, CELI, non enregistré, etc.)
- [ ] **Rendement annualisé 5 ans** (idéal pour comparer)
- [ ] Sinon : rendement 1 an, 3 ans, et/ou depuis l’ouverture
- [ ] Date du chiffre (ex. « au 30 juin 2026 »)
- [ ] Série / catégorie de parts si indiquée (A, Investor, Conseiller…)

## Où ça se trouve souvent

| Document | Ce qu’on y trouve |
|---|---|
| Écran « Rendement / Performance » dans l’app | Chiffres rapides (1 an, YTD, parfois 3–5 ans) |
| Relevé de placements / portefeuille | Avoirs + parfois performance par compte |
| **Rapport annuel sur le rendement des placements** | Taux personnalisé 1 an + depuis l’ouverture (obligatoire CRM2) |
| Feuillet / fiche du fonds | Rendement du fonds (pas toujours le vôtre personnel) |

## Astuce pour la comparaison

Pour comparer à profil égal avec les Portefeuilles Modèles iA, on cherche idéalement :

> « Rendement net annualisé sur **5 ans** » du même style de risque  
> (prudent / modéré / équilibré / croissance / audacieux)

Si seul le **rapport annuel** est disponible (1 an + depuis ouverture), on peut quand même faire un premier portrait — apportez-le.

## Confidentialité

Masquez numéro de compte, NAS, adresse si vous partagez une capture. Le **nom du fonds + les % de rendement** suffisent souvent pour démarrer.
