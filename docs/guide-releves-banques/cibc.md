**Hors portée de ce guide :** CIBC Investor’s Edge / Pro-Investisseurs (courtage autonome).

---

## Contexte

Les fonds achetés via le réseau CIBC / **Service Investisseurs Impérial (Imperial Investor Service)** ont des relevés et un **rapport annuel de rendement** (pourcentage annualisé, dépôts/retraits).

---

## Parcours documents (texte CIBC)

1. Se connecter à **Imperial Investor Service** (ou espace placements CIBC lié)  
2. **Accounts** → **eDocuments**  
3. Choisir le compte  
4. Télécharger :
   - relevés de compte  
   - **rapport annuel sur le rendement**  
   - documents fiscaux (selon le cas)  

Documents conservés jusqu’à **7 ans** en ligne.

App CIBC Mobile Banking : menu **Plus** → documents / relevés pour les comptes liés.

---

## Quoi envoyer

- PDF du **rapport annuel de rendement**  
- Nom du fonds CIBC
